We can treat missing values in the following ways -
1 - Ignore them
2 - Replace them with mean if the feature is continous
3 - Replace them with the most occured value if the feature is categorical
4 - Use a model to predict the missing values

drive_unit and segment are categorical variables, replacing the missing values with most frequently occured value will result in a bias. So removing the missing values.

# A function to deal with categorical varibales
def categ_to_int(cars, column):
    #Creating dummy varibales
    col = pd.get_dummies(cars[column], drop_first = True)
    #Concatenating them together
    cars = pd.concat([cars, col], axis = 1)
    #Droping the original column
    cars.drop([column], axis = 1, inplace = True)
    return cars
